# 更新履歴

MeteoScope の主な更新履歴です。  
日付は Git 履歴を基準に、利用者から見える変更を中心にまとめています。

## 2026-07

### アプリ名・公開基盤

- アプリ名とリポジトリ名を `MeteoScope` に変更。
- Cloudflare Pages 配信に対応し、`wrangler.toml`、Pages Functions、KV バインディング設定を整備。
- アプリのアイコンを MeteoScope 用デザインに変更。

### 管理者機能

- `/admin.html` にパスワード保護付きの管理者画面を追加。
- お知らせの追加、編集、削除に対応。
- 管理者画面から登録したお知らせを、アプリ上部にテロップとして表示できるように変更。
- 利用者意見フォームを設定モーダルに追加し、管理者画面で確認できるように変更。
- 管理者画面のスマホ、タブレット表示を改善。
- 設定モーダル内の項目を折りたたみ式に変更。

### モバイル UI

- スマホ縦画面の下部タブを Liquid Glass 風のボタンバーに変更。
- 下部ボタンバーをスライド操作できるように改善。
- 要約バーと詳細パネルを追加し、スマホ縦画面で地図を見ながら主要情報を確認できるように変更。
- 要約バー、詳細パネル、下部ボタンバーのガラス質デザインを統一。
- タブごとの要約バー高さを統一し、読み込み中でもレイアウトが崩れにくいように改善。
- タブ切り替え時にアプリが落ちたり、古い UI に戻ったりする問題を軽減。
- 現在地ボタン、凡例ボタン、地図上ボタンの配置をスマホ表示向けに調整。
- 下部ボタンの並び替えを設定モーダルから変更できるように追加。

### 雨雲レーダー・天気図

- 降水ナウキャストを現在地直下の値として読み取る表示を追加。
- 雨雲レーダーの要約バーから時刻スライダーを操作できるように変更。
- 要約バー上でもスライダー操作中に時刻と地図表示が追従するように改善。
- 雨雲レーダータブに天気図表示を追加。
- 天気図は雨雲レーダーと切り替えて表示できるようにし、表示中は雨雲レーダーを非表示に変更。
- 天気図の実況、予想をスライダーで切り替えられるように追加。
- 天気図を 2 日前まで表示できるように拡張。
- 天気図の高気圧、低気圧、台風ラベルを表示し、中心気圧も併記するように改善。
- 寒冷前線、温暖前線、閉塞前線、停滞前線の描画を追加。
- 天気図スライダーの時系列順、日付、地図反映の不具合を修正。
- 天気図表示中の横線ノイズや過剰な拡大表示を抑制。

### アメダス

- アメダス要約バーに、現在地に最も近い観測点の情報を表示。
- 気温、降水量、風速、積雪量の色分けと凡例を調整。
- 気温ランキングを 20 位まで表示できるように変更。
- ランキングの高い順、低い順を切り替え可能に変更。
- ランキング項目を押したとき、該当観測点へ地図をフォーカスするように追加。
- 気温や降水量など値が大きい観測点が地図上で上に表示されるように調整。
- エラーになっている観測点を表示しないように変更。

### 警報・注意報・早期注意情報

- 現在地の警報、注意報、危険警報、特別警報の通知機能を追加。
- 新規発表、継続、切替、解除を区別して通知するように改善。
- 注意報のみの新規発表通知は、設定でオン、オフできるように変更。
- 警報タブの発表状況、早期注意情報、キキクル切り替えを要約バー内に移動。
- 早期注意情報選択時に、現在地の発表状況を要約バー、詳細パネルへ表示できるように変更。
- 早期注意情報の 4 日後、5 日後の表示漏れを修正。
- 発表がない日も `-` として表に表示するように変更。
- 警報・注意報の文字色を、注意報は黒、それ以外は白に統一。
- 今後の見通しで `暴風` と表示されていた箇所を `強風` 表記へ修正。
- 警報タブの読み込み、キキクル切り替え、今後の見通し表示のパフォーマンスを改善。
- 市区町村の地図色塗りが拡大縮小中に荒くなる問題を軽減。
- 愛媛県西条市、青森県青森市、大分県別府市など、一部市区町村で拡大時に色が抜ける問題を修正。

### 台風情報

- 台風の現在位置、過去経路、予報経路、予報円、強風域、暴風域、暴風警戒域を表示。
- 複数台風がある場合、台風選択を 1 つのボタンで切り替えられるように変更。
- 台風要約バーに中心気圧と最大瞬間風速を表示。
- 台風選択時に、予想進路も含めて地図へフォーカスするように改善。
- 予報円をタップすると、対象予報円の情報を表示できるように追加。
- 台風の大きさ、強さ、中心気圧、移動、最大風速、最大瞬間風速の表示を整理。
- 勢力が弱まって強さ情報がなくなった場合、古い強さが残らないように修正。
- `台風第0a号` ではなく `熱帯低気圧a` と表示するように修正。
- 暴風警戒域の描画を繰り返し改善し、線の途切れ、二重線、交差、片側欠けを軽減。

### 地震情報

- 地震タブを追加し、気象庁 XML から地震情報を取得。
- 最新地震をデフォルト表示し、過去地震は履歴から選択できるように追加。
- 地図上に各地の震度と震源を表示。
- 震源は `×` で表示し、震源名や震度ラベルの地図上表示は廃止。
- 地震履歴を約 11 件表示できるように変更。
- VXSE51、VXSE52、VXSE53 をイベント ID と発生時刻で同一地震としてまとめるように改善。
- 地震情報のリアルタイム更新を追加。
- 震源地、発生時刻、最大震度、マグニチュード、深さを詳細パネルに表示。
- 深さ 0 km は `ごく浅い` と表示するように変更。

### 防災マップ PDF

- 地震ボタンを 3 回押す隠し操作で、防災マップ PDF ビューアを開けるように追加。
- PDF ファイル選択方式に限定し、表示領域を広く確保。
- PDF のズーム、スクロール、スマホ表示を改善。
- 選択した PDF を再起動後も保持できるように変更。
- 設定モーダルから保存済み PDF を削除できるように追加。

### 地図・共通 UI

- MapLibre GL ベースの暗色地図へ移行。
- 全タブで市区町村区分を表示。
- 凡例を折りたたみ式に変更し、起動時は閉じた状態に統一。
- 凡例ボタンをコンパクト化し、地図上の配置を調整。
- 凡例が画面外にはみ出る問題を修正。
- 市区町村境界や警報色塗りの描画安定性を改善。
- アプリ全体の日本語フォント、タイポグラフィを調整。
- 自動情報更新機能を追加。

## 2026-06

### 初期実装

- Vite ベースの静的 Web アプリとしてプロジェクトを作成。
- アプリ HTML、エントリポイント、タブ制御、時計、左パネル、地図の基本構成を追加。
- 気象庁データ取得クライアントと各種設定を追加。
- Leaflet から始まり、後に MapLibre GL へ地図実装を拡張。

### 雨雲レーダー

- 気象庁降水ナウキャストの表示を追加。
- 更新時刻表示、前後時刻移動、再生、現在時刻ボタンを追加。
- 3 時間前まで遡って表示できるように拡張。
- 雨雲レーダーの凡例をアメダス降水量と統一。

### アメダス

- 気象庁アメダス JSON から、気温、降水量、風速、積雪量を取得。
- 観測点を地図上に丸で表示。
- 降水量、気温、風速、積雪量の色分けを調整。
- 風速は風向矢印で表示するように変更。
- 選択項目のランキング表示を追加。

### 警報・注意報

- 気象庁の警報・注意報データを取得し、市区町村単位で地図に反映。
- 注意報、警報、危険警報、特別警報の色分けを追加。
- 特別警報は紫色に黒斜線を重ねる表現を追加。
- 発表中の市区町村リストと地図選択を連動。
- 市区町村をタップすると詳細モーダルを開けるように追加。
- 今後の見通し表示を追加。
- 河川氾濫、大雨、土砂災害、高潮に関するもののみレベル表記するように調整。
- 早期注意情報の表示を追加。
- 土砂キキクル、浸水キキクル表示を追加。
- 洪水キキクルと河川レイヤーは検証後にいったん廃止。

### 台風情報

- 台風情報タブを追加。
- 台風が発生していない場合は `現在、台風情報は発表されていません` と表示。
- 気象庁台風 JSON から現在位置、予報円、暴風域、強風域を取得。
- 過去経路表示を追加。
- 予報円と暴風警戒域の共通外接線描画を追加。
- 台風の暴風警戒域描画を、実データや過去電文で検証しながら調整。
- 5 日予報がない電文でも表示が崩れないように対応。

### 位置情報

- 現在地取得機能を追加。
- 現在地の警報・注意報を表示。
- マイエリア機能を追加。
- 自動で位置情報を取得し続ける設定を追加。
- 現在地周辺の雨雲、警報・注意報、マイエリア情報を表示。

### UI・デザイン

- EQ-app-2026 に近いダークテーマ、左パネル、地図表示を導入。
- モバイル向け下シート表示を追加。
- 左パネル、凡例、時計、ボタン、アイコンのデザインを調整。
- アプリ内の各種説明文、見出し、ボタンのサイズと配置を整理。
- 設定モーダル、利用規約、プライバシーポリシーを追加。

### デプロイ

- GitHub Pages 用のデプロイ設定を追加。
- Cloudflare Pages 移行に向けた設定を追加。
- README を更新し、機能、データ取得元、開発、ビルド、デプロイ手順を整理。
